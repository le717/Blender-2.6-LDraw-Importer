System Requirements

  • Blender 2.63 or higher
  • ldraw_import.py (included with this download)
  • LDraw System of Tools
  • MLCad, LeoCAD (both optional)

Installation

Windows, Mac OS X, Linux

  • Open Blender, and click File > User Preferences > Addons > Install from File...
  • Browse for ldraw_import.py, and open it.
  • Find the script name in the list of Addons (search for LDraw if needed), and tick the check mark to activate it.
  • If you want to load the script at all times, click the Save as Default button.

Setting the LDraw Folder Path

  • Open Blender, and click File > Import > LDraw Model (.dat/.ldr)
  • In the bottom left corner of your screen, under the Import LDraw Model header, there is a small field with the title LDraw Path. Modify the field with the proper folder path to your LDraw System of Tools installation path.
  • Click the + about the path and name the preset to save your installation path. You will select the present each time you open Blender to set your installation path to the LDraw System of Tools.

That's it! You have successfully installed the Blender 2.6 LDraw Importer script.

Known Bugs


  1. Bricksmith LDR and MPD files do not import properly
  2. Entire import stops upon reaching an non-existent brick
  3. Blender crashes if user imports multiple models and available RAM is too low to continue. Can never be fixed, due to various computers with different amounts of RAM
